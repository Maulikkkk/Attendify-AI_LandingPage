# 🚀 Attendify AI – Landing Page

A modern, responsive landing page for **Attendify AI**, an AI-powered attendance system that uses **face recognition and voice biometrics** to automate classroom attendance.

---

## 🌐 Live Demo

👉 https://attendifyai.streamlit.app/

## 📌 About This Repo

This repository contains the **landing page frontend + Flask integration layer** for Attendify AI.

```bash

cd Attendify-AI_LandingPage

pip install -r requirements.txt
python app.py
```

Then open:

```
http://127.0.0.1:5000
```
